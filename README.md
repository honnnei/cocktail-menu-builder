### Tech Stack
##### API:
* The Cocktaildb API https://www.thecocktaildb.com/
##### Backend Server:
* Node JS
* Express JS
##### Database:
* MongoDB + Mongoose
##### Frontend:
* TypeScript 
* React JS with Hooks (create-react-app --template typescript)
* React Router
##### UI:
* Bootstrap 4.5
